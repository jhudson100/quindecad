View Controls
  • Rotate: Left mouse
  • Pan: Shift+Left mouse OR Unshifted Middle mouse
  • Zoom (dolly): Right mouse OR wheel
  • g: Show or hide grid
Run Code
Type Python code in the edit area. To run the code, press shift+enter or click the "Run" button.
Code
Create one or more drawable objects and send them to the draw() function. For example: 
c = cube(xsize=3,ysize=2,zsize=1,centered=True,color=(255,255,0))
s = sphere(x=0, y=0, z=0, radius=1.5)
d = difference([c,s])
draw(d)
Functions
Many modules from the standard Python library are available and can be imported with "import" (ex: "import math").
Some additional functions are available. In the following descriptions, a color refers to a list or tuple of three values from 0 to 255 inclusive. A fourth element may be specified; it represents the opacity (255=fully opaque; 0=fully transparent)
