View Controls
Mouse controls:
  • Left mouse: Rotate
  • Shift+Left mouse: Pan
  • Middle mouse: Pan
  • Right mouse: Zoom (Dolly)
  • Wheel: Zoom (Dolly)
There are also some key controls available for the view. If the Python panel has keyboard focus, you'll need to click on the view panel first so it will be active and will receive the keystrokes.
  • g: Show or hide grid
  • a: Show or hide axes
  • p: Print the coordinates beneath the pointer (the pointer must be over an object)
  • ←, →, ↑ ↓: Pan
Run Code
Type Python code in the edit area. To run the code, press shift+enter or click the "Run" button.
Code
Create one or more solid objects and send them to the draw() function. For example: 
c = cube(xsize=3,ysize=2,zsize=1,centered=True,color=(255,255,0))
s = sphere(x=0, y=0, z=0, radius=1.5)
d = difference([c,s])
draw(d)
