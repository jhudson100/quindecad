View Controls
  • Rotate: Left mouse
  • Pan: Shift+Left mouse OR Unshifted Middle mouse
  • Zoom (dolly): Right mouse OR wheel
  • g: Show or hide grid
Run Code
Type Python code in the edit area. To run the code, press shift+enter or click the "Run" button.
Code
Create one or more drawable objects and send them to the draw() function. For example: 
c = cube( xsize=4, ysize=3, zsize=2, centered=True )
s = sphere( x=0, y=0, z=0, radius=2 )
d = difference(c,s)
f = frustum(
    x=5,y=0,z=0, #center
    radius1=2,      #bottom radius
    radius2=0.25,   #top radius
    height=4,      #height
    zcenter=True,
    color=(255,128,0) 
)
draw(d,f)
Functions
Many modules from the standard Python library are available and can be imported with "import" (ex: "import math").
Some additional functions are available. Each one returns a drawable object. Some functions take a color argument. A color is specified as a list or tuple of three values from 0...255 inclusive. If a fourth element is present, it represents the opacity (255=fully opaque; 0=fully transparent)
